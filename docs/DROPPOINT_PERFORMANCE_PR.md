### Requirements for Contributing a Performance Improvement

* Fill out the template below. Any pull request that does not include enough information to be reviewed in a timely manner may be closed at the maintainers' discretion.
* * Include `[Performance]` in the Pull Request title.
* The pull request must only affect performance of existing functionality. To contribute other changes, you must use a different template. You can see all templates at <https://github.com/GameGodS3/DropPoint/tree/main/docs/PULL_REQUEST_TEMPLATE.md>.
* After you create the pull request, all status checks must be pass before a maintainer reviews your contribution.
